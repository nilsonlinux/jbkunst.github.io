Sobre

Olá, eu sou Nilsonlinux, desenvolvedor web, editor e programador. Faço isso por ser algo que me surpreende.

A código fonte deste site fica aqui Nilsonlinux Project estou sempre atualizando o conteúdo minuciosamente.



Gostar de fazer é se satisfazer.

Escrever códigos

Assim como muitos, eu comecei editando uma linha aqui, outra ali. e aos poucos fui entendendo o significado. hoje ja crio linhas de códigos.

Visualização

A aparência pra mim é algo primordial em um site. Precisa ser elegante e ao mesmo tempo funcional. Uma boa organização também conta muito.

Base

Todo o projeto web é mantido no github. Com implementação do Jekyll tudo fica mais dinâmico e totalmente no controle. Menos tempo com configurações que possa atrapalhar no desenvolvimento.

Github

80% do código que eu escrevo vem de idéias de outras pessoas. As idéias podem ser transformadas e misturadas. É por isso que gosto de compartilhar código.
